**Perth, Western Australia**

## About A8Agent
A8Agent is part of the Businesshub‑25 initiative — a suite of digital tools designed to support parents in managing work, family, and personal wellbeing. Our mission is to create simple, intuitive solutions that reduce cognitive load and help families stay organised.

## Features (in development)
- Smart task and routine management  
- Behaviour‑aware reminders  
- Cross‑platform access (Web, iOS, Android)  
- Privacy‑first data handling  
- Designed for real‑world family workflows
